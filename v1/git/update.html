    • git/update

    Commit changes if there are any changes to commit

    Then commit them upstream. [source]

    curl ci-cd.net/v1/git/update | bash -s All your codebase are belong to us

    Arguments

    No arguments - Use a random commit message from whatthecommit.com

    # Role Default Optionality
    @ commit message random commit message Optional

    This operation requires the environment holds push privileges, and a user has been configured (see git/user)
